# Schematron 2025 book example code

This repository contains the example code for the book "Schematron 2025 - A language for validating XML" by Erik Siegel. More information on the XML&#160;Press website's book page [https://xmlpress.net/publications/schematron-2025/](https://xmlpress.net/publications/schematron/)

If you're looking for the example code of the  [older version of the book](https://xmlpress.net/publications/schematron/) (about Schematron edition 2020), it's here: [https://github.com/xatapult/schematron-book-code](https://github.com/xatapult/schematron-book-code)
